# TabStacks

A Chrome extension that automatically organizes your open tabs into Chrome Tab Groups by domain, calling them "stacks", and injects a lightweight switcher bar so you can jump between them instantly.

## Features

- Auto-groups tabs by domain into collapsible stacks
- Switcher bar overlay for visual tab navigation; keyboard shortcuts work without it
- Runs entirely in your browser — no data ever leaves your device
- Pro license support via Gumroad (optional)

## Privacy

TabStacks does not collect, transmit, or sell any personal data. All tab and group state is stored locally on your device.

See the full [Privacy Policy](https://tabstacks.github.io/web/privacy-policy.html).
